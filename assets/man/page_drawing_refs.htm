Sketch references

This dialog is opened by the Reference button of the Sketch window.

  • legs, survey midline (white)
  • splays (grey)
  • stations names (violet)
  • grid, metric or yard
  • scalebar, reference scalebar
  • outline, outline of another sketch
There are three grids,
  1. a grid at 1 m spacing (dark grey)
  2. a grid at 10 m spacing (grey)
  3. a grid at 100 m spacing (light grey)

TopoDroid shows only the grids that are appropriate for the scale of the sketch view.

If there are other sketches, the outline of another sketch can be displayed (in grey) to help drawing the sketch. Checking the box opens a new dialog to pick the sketch to display, or to clear the outline display.

A further checkbox [E] specifies whether the reference should be held fixed when shifting the drawing with two fingers. This is used to adjust a PocketTopo imported drawing to the reference in the case it gets imported with a translation.
